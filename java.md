# 변수 선언

`int a;` 이게 변수를 선언한다는 뜻이고 `a = 10;` 이게 변수 초기화라는 뜻이다.

# 변수

- int를 한번만 적어도 여러개 선언을 할 수 있다고 정수를 할 때 사용한다.
- double은 실수를 할 때 쓰고
-  char은 문자 하나를 할 때 쓰고
-  String은 문자열을 할 때 쓴다.
- 실수를 나타낼 때 쓰는 것 float, double가 있는데 float는 저장하는 수 뒤에 f를 붙여야 한다.

# 산술 연산자

* 덧셈을 할 때 `+`
* 뺄셈을 할 때 `-`
* 곱셈을 할 때 `*`
* 나눗셈을 할 때 `/`
* 나머지는 `%`

# 증감 연산자

`++`은 1 증가시키라는 뜻 `--`은 1 감소시키라는 뜻.

# 전위 증감 연산자

ex) `b = ++a; //a`의 값을 먼저 증가시키고, 그 결과를 b에 대입
`System.out.println("a = " + a + ", b = " + b);`

# 후위 증감 연산자

ex) `b = a++; //a`의 현재 값을 b에 먼저 대입하고, 그 후 a 값을 증가시킴  
`System.out.println("a = " + a + ", b = " + b);`

# 비교 연산자

* `==` 동등성
* `!=` 불일치
* `>` 크다
* `<` 작다
* `>=` 크거나 같다
* `<=` 작거나 같다.

# 비교 연산자

비교 할 때 쓰는 함수 equals. boolean은 무조건 true나 false가 나오게 함.

# 논리 연산자

* `!`는 부정하는 뜻. 만약 false면 true를 출력함
* `&&`는 true가 둘 다 있어야 true로 출력. 하나라도 false가 있으면 false 출력
* `||`는 하나만 true여도 true로 출력. false를 출력하려면 다 false여야 함.

# 대입 연산자

* `+=`는 더하고 대입
* `-=` 빼고 대입
* `*=`곱하고 대입
* `/=` 나누고 대입
* `%` 나머지 대입.

# if문

if문은 특정 조건이 참인지 확인하고 그 조건이 참(true)일 경우에만 특정 블록을 실행한다. 

# else if

else if 문은 앞선 if 문의 조건이 거짓일 때 다음 조건을 검사한다. 만약 앞선 if 문이 참이라면 else if를 실행하지 않는다.

# switch

문은 if문을 조금 더 편리하게 사용할 수 있는 기능.

# 삼항 연산자

`String a = (x % 2 == 0) ? "짝수" : "홀수";`
// 조건 | 참 | 거짓
조건에 짝수가 정답이면 그대로 출력, 틀리면 홀수 출력.

# while문

* 조건식을 확인한다. 참이면 코드 블럭을 실행하고, 거짓이면 while문을 벗어난다.
* 조건식이 참이면 코드 블럭을 실행한다. 이후에 코드 블럭이 끝나면 다시 조건식 검사로 돌아가서 조건식을 검사한다.

# do-while문

while문과 비슷하지만 조건에 상관없이 무조건 한 번은 코드를 실행한다.

# break, continue

break는 반복문을 즉시 종료하고 나간다. continue는 반복문의 나머지 부분을 건너뛰고 다음 반복으로 진행하는데 사용된다.

# for
   ```
   for (int i = 1; i <= 10; i++){
             System.out.println(i);
```

while문과 같은 반복이고, 코드를 반복 실행하는 역할은 한다.  
for문은 주로 반복 횟수가 정해져 있을 때 사용한다.  
`for(초기식; 조건식; 증감식)`
`for(;;)` 이렇게 생략 가능.   
왜냐하면 초기식, 조건식, 증감식은 선택이기 때문에.

```

        students = new int[]{90, 80, 70, 60, 50};
```
동일한 데이터 타입의 변수 여러 개를 메모리상에 연속적으로 나열하여 하나의 묶음으로 다루는 정적 자료구조


# 향상된 for 문

```

```
